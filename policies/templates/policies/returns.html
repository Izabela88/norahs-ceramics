RETURNS

All returns must be postmarked within fourteen (14) days of the purchase date. All returned items must be in new and unused condition, with all original tags and labels attached.

To return an item, please email customer service at studycode88@gmail.com to obtain a Return Merchandise Authorisation (RMA) number. After receiving a RMA number, place the item securely in its original packaging and the return form provided, then mail your return to the following address: Norah's Ceramics Attn: Returns RMA # 3 Castle Hill Walk Bridgnorth, West Midlands WV16 4AF United Kingdom Please note, you will be responsible for all return shipping charges. We strongly recommend that you use a trackable method to mail your return.
